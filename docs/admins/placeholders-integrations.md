# Placeholders And Integrations

HavenNecessities has soft integrations with PlaceholderAPI, VaultUnlocked, and FancyDialogs.

## PlaceholderAPI

Balance placeholders:

- `%havennecessities_balance%`
- `%havennecessities_balance_raw%`
- `%havennecessities_currency_name%`

Profession placeholders:

- `%havennecessities_profession_main%`
- `%havennecessities_profession_main_id%`
- `%havennecessities_profession_main_level%`
- `%havennecessities_profession_main_xp%`
- `%havennecessities_profession_hobby%`
- `%havennecessities_profession_hobby_id%`
- `%havennecessities_profession_hobby_level%`
- `%havennecessities_profession_hobby_xp%`

## VaultUnlocked

VaultUnlocked bridge scaffolding charges against HavenNecessities wallets when used by integrations. It does not own balances.

## FancyDialogs

FancyDialogs can provide richer profession and cosmetics UIs. Chat fallback is used when dialogs are unavailable.
