# Profile

The profile module shows a summary of a player's HavenNecessities progress when enabled.

## Commands

| Command | Description |
| --- | --- |
| `/profile` | View your own profile summary. |
| `/profile <player>` | View another player's profile when permitted. |
| `/profile <player> <category>` | Open a specific profile category. |

Profile summaries can include economy, professions, attributes, storage, flight, Multi-Tool, cosmetics, and tracked stats.

## Examples

```text
/profile
/profile Steve
/profile Steve professions
```

## Permissions

- `havennecessities.profile.view`
- `havennecessities.profile.view.others`
