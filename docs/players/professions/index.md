# Professions

Professions let players earn XP, income credit, storage growth, and stat perks from themed gameplay.

Players can choose one main profession and one hobby profession. Main professions receive full rewards. Hobby professions receive reduced rewards.

## Reward Roles

- Main: 100% XP, income, storage growth, and stat perks.
- Hobby: 35% XP, 35% income, no storage growth, and 35% stat perks.
- Payouts: enabled every 300 seconds when configured income is available.

## Commands

| Command | Description |
| --- | --- |
| `/professions` or `/profession` | Open profession selection. |
| `/profession <profession>` | Open a profession detail view. |
| `/professions list` | List all professions. |
| `/professions status` | Show selected professions, levels, XP, progress, and pending payout. |
| `/professions select main <profession>` | Choose a main profession. |
| `/professions select hobby <profession>` | Choose a hobby profession. |
| `/professions leave <main|hobby>` | Clear one selected role. |
